* 下载地址 [https://github.com/eleven26/tw-tools/releases/latest](https://github.com/eleven26/tw-tools/releases/latest)
* 下载 *.zip 文件, 解压
* [自定义插件安装方法](https://www.jetbrains.com/help/phpstorm/installing-plugin-from-disk.html)
* 安装的时候选择上面 zip 文件
